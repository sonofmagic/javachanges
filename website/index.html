javachanges

A small Java CLI for Maven monorepos that want Changesets-style release planning, generated changelogs, and CI-friendly publish workflows.

View Repository Read README Getting Started

File-based release intent

Track release notes in `.changesets/*.md` and keep reviewable release metadata in git.

Maven-focused

Built for multi-module Maven repositories using a root `revision` and release-plan updates.

Automation-friendly

Includes preflight, publish, release notes, and GitHub/GitLab variable management helpers.
